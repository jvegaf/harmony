## Key Technical Discovery
Lofty crate frame type mapping:
- Text frames (T***) → ItemValue::Text → .text()
- URL frames (W***) → ItemValue::Locator → .locator()
- Binary frames → ItemValue::Binary → .binary()

WOAF is a URL frame, requires .locator() not .text().
